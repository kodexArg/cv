# cv

Personal CV — Astro + Svelte 5 + Tailwind v4. Deployed to GitHub Pages at `/cv`.

- `/cv/` — one-page CV (English): AI Solutions Architect · Fractional CTO. Pure static Astro, prints to a single A4, schema.org `Person` JSON-LD.
- `/cv/full/` — legacy full CV (Spanish), kept for reference.

```sh
bun install && bun dev
```
